springboot_admin_server应用监控管理

前言

Spring Boot Admin 用于监控基于 Spring Boot 的应用，可对应用信息可视化，提供应用监控接口管理，方便用户管理操作。
运行环境：maven，spring boot应用， Spring cloud eureka的配置

1.在 Spring Boot 项目中配置

首先，先建个基础的spring boot 应用，该应用作为 Spring Boot Admin Server，其他的spring boot应用作为client，当把client应用注册到 server时，我们就能在server服务管理上可以看到注册的client应用状态信息了。
